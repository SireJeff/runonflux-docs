# Security

1. **How do you safeguard my data from unauthorized access by other FluxEdge users?** FluxEdge employs encryption and access control measures to ensure data security and privacy.
2. **What safeguards are in place to protect my data from FluxEdge GPU providers?** Data protection protocols and secure data handling practices are enforced to protect user data from providers.
3. **What measures are in place to ensure the security of my credit card details when using FluxEdge?** FluxEdge uses secure payment gateways and encryption to protect credit card details.
4. **Will my IP address be exposed to GPU providers or other users on FluxEdge?** No, FluxEdge ensures that your IP address remains hidden from providers and other users.
