# Projects

The Audit Projects tab provides a complete history of project deployments associated with your account.

You will see a data table with detailed information about all your project deployments.

* Date: The date and time of the action, with an info button showing creation and end timestamps.
* Project: The deployed project, including its icon (if applicable), name, and service type (e.g., artificial intelligence, rendering, cloud gaming).
* Duration: The duration of the deployment.
* Machines: The number of machines on which this project was deployed.
* Price per Hour: The hourly rate paid for the deployment.
* Total Spent: The total amount spent on this deployment.
