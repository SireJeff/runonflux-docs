# How to Set Up an Anti-Phishing Code on FluxEdge

**What is an Anti-Phishing Code and Why is it Important?**

An anti-phishing code is a personalized identifier that helps protect you from phishing attacks. When set up, this code appears in all official emails from FluxEdge, allowing you to verify the legitimacy of the communication. It ensures that any email you receive is genuinely from FluxEdge and not from malicious sources attempting to steal your information.\
\
**Step-by-Step Guide to Set Up Your Anti-Phishing Code**

1. Sign In:

* Log in to your FluxEdge account at [alpha.fluxedge.ai](http://alpha.fluxedge.ai).\


2. Navigate to Settings:

* Go to your Account section and then click on Settings.

3. Access Security Settings:

* In the Settings menu, select the Security tab.

<img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXcabR5sJKdA8ZMBwSkLrA28YMyozKRCctv26ayDr6zbC6-0IiieTiqTwwBv7VmcwLEAr_3pViOyrQ1f3N7DgnXFQbeIENc333G-whkBe4kscc8XTDqINIVEeMQ5DGjARwgLi1kYzTo2bM3RnefEnBG5KWcw?key=jA6g72zw6HGt1yIkI6x15g" alt=""/>

4. Enable Anti-Phishing Code:

* Under Advanced Security, locate the Enable button next to the Anti-Phishing Code option.

5. Understand the Anti-Phishing Code:

* A pop-up will appear explaining what an anti-phishing code is and its importance. After reading, click on Create Anti-Phishing Code to proceed.

<img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXdBN2507HIArejD0tv_mwPb7g0HEeAhR9GfrtPpO-fR7rWqcX7kScCtw3NBJiL2Tq0Ozo-EmgS1LDJBi9wxdzU1375lN53FajfXB0YGQRNbLQZi4P38T2ohYHyJBSnJ2oIDR289nZnQ_Lev1iXHGojmFBN1?key=jA6g72zw6HGt1yIkI6x15g" alt=""/>

6. Create Your Code:

* Enter your desired code, which should be between 4 to 20 characters long and must not include any special characters. After entering your code, click Submit.

<img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXfFnDuBj1_CC7Kp7MseLCQbnwHlBePQdD8fru9fkoQlazEiIOM1CxzKmMZKMGBVqB3pL2liBcdsHQGDAHlkfO4uCHos6XB9BII6vNnRhx-N2wpjqE4BFtY4xugUtJNdvR1XsyCnXbuS3xdR_usWCuxr7YX0?key=jA6g72zw6HGt1yIkI6x15g" alt=""/>

7. Confirmation:

* Your anti-phishing code is now set. It will appear in all email notifications from FluxEdge, helping you verify the authenticity of the communications you receive.

By setting up an anti-phishing code, you add an extra layer of security to your account, ensuring that all communications from FluxEdge are genuine.
