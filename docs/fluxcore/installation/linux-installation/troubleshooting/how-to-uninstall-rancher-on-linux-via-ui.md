# How to Uninstall Rancher on Linux via UI

#### 1. Access the Service Module

Click the **Services** tab in the menu bar.

<img src="/.gitbook/assets/image (61).png" alt=""/>

#### 2. Leave the Cluster

* at **My Machines**, find the machine you want to remove.
* Click the **Settings** icon for that machine.
* In the pop-up, go to **Step 2: Choose Your Cluster** and unselect the cluster.
* Confirm the action.
