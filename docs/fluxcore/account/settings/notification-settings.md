# Notification Settings

This page allows you to configure your email notifications in the FluxCore platform. These settings help you stay informed about your FluxCore activities, even when you're away from the local computer. Each notification option is initially disabled by default and can be activated individually based on your preferences. Below is a detailed overview of each notification type:\


<img src="/.gitbook/assets/Overview (62).png" alt=""/>

**1. Withdrawal Successful**

* **Description**: You receive a notification when one of your FluxCore withdrawal requests has been successfully processed by the InFlux team.
* **Purpose**: This alert confirms that your withdrawal has been completed, providing you with the transaction amount and an explorer link for tracking.

**2. New FluxCore Update**

* **Description**: You receive a notification when a new FluxCore update has been released, along with its patch notes.
* **Purpose**: This notification keeps you informed about the latest improvements and features available in FluxCore, helping you stay up-to-date with the software.

**3. Machine Lease Started**

* **Description**: You receive a notification when one of your machines has successfully been leased via the FluxEdge Marketplace and is earning money.
* **Purpose**: This alert informs you when a machine begins generating revenue, including the machine's identifier (last 10 digits of its machine ID) and the current hourly earnings.

**4. Machine Lease Stopped**

* **Description**: You will receive a notification when a machine that was leased out has its rental stopped. This could happen if the FluxEdge customer completed the rental, if there were provider-side malfunctions (e.g., network issues), or if you manually forced the lease to stop.
* **Purpose**: This notification informs you about the end of a lease, detailing the total rental earnings and indicating if any penalties apply due to an early termination.

**5. Machine Unavailable on FluxEdge**

* **Description**: You will receive a notification if one of your machines, which was available for lease on FluxEdge, becomes unavailable and is removed from the cluster.
* **Purpose**: This alert helps you quickly address connectivity or outage issues, providing details about the affected machine (last 10 digits of the machine ID) and offering guidance on restoring its availability for future leases.

#### **Anti-Phishing Code Information**
