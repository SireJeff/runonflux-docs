# How to Set Up an Anti-Phishing Code on FluxCore

\
**What is an Anti-Phishing Code and Why is it Important?**

An anti-phishing code is a personalized identifier that helps protect you from phishing attacks. When set up, this code appears in all official emails from FluxCore, allowing you to verify the legitimacy of the communication. It ensures that any email you receive is genuinely from FluxCore and not from malicious sources attempting to steal your information.\


**Step-by-Step Guide to Set Up Your Anti-Phishing Code**

1. Sign In:

* Launch the FluxCore application and log in to your account.

2. Navigate to Settings:

* Go to your Account section and then click on Settings.

3. Access Security Settings:

* In the Settings menu, select the Security tab.

4. Enable Anti-Phishing Code:

* Under Advanced Security, locate the Enable button next to the Anti-Phishing Code option.

5. Understand the Anti-Phishing Code:

* A pop-up will appear explaining what an anti-phishing code is and its importance. After reading, click on Create Anti-Phishing Code to proceed.

<img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXd6a-8OlkAgjUh0kfV5gnJZCEmCxJNEcPW_jRoICeI54umvlgQ1d8D7cEhtVRSpxO3L0i34b5XJIz9Hr3oT96oEndeForiKuh5syfcWKb9wFBym4_ZFgoIwIKst_WQqTIJBW7XCSBA-Pprg5NNIewVye3s?key=9a9nCv2MIF0FzMSJDBd05A" alt=""/>

6. Create Your Code:

* Enter your desired code, which should be between 4 to 20 characters long and must not include any special characters. After entering your code, click Submit.

<img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXdOuwy_fL9DnCu0as-g4KAB3EqAx7gD6iYJpxracuRmXIj_0_16l8k_xuzparKZDMVrkIhRAJ-D6Y6PeKzTxg5mSpYvYlsQVCfP09svDfJh_mlbcz4-bh8y2a_-7Sdy111vFifd1hwEVmlMDIdwIziRceE?key=9a9nCv2MIF0FzMSJDBd05A" alt=""/>

7. Confirmation:

* Your anti-phishing code is now set. It will appear in all email notifications from FluxEdge, helping you verify the authenticity of the communications you receive.

By setting up an anti-phishing code, you add an extra layer of security to your account, ensuring that all communications from FluxEdge are genuine.
