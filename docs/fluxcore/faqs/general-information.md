# General Information

1. **What are the system requirements for joining FluxCore?**

* **Supported Operating Systems:** Windows 10 and 11, and Linux Ubuntu 22.04.
* **Minimum:** 1 CPU, 1 Core, 1 Thread, 12GB RAM, 100GB storage
* **Recommended:** 1 CPU, 4 Cores, 4 Threads, 16GB RAM, 100GB storage
* **Minimum Download/Upload Speed:** 1 Mbps
* **Recommended Download/Upload Speed:** 50 Mbps / 200 Mbps

2. **Do I need to install any software to participate in FluxCore?** Yes, you need to install the FluxCore client, which allows you to benchmark your machines, participate in POW mining, and make your machine available for rentals on FluxEdge.
3. **Can I rent out my GPU if I'm using it for other tasks occasionally?** Yes, you can rent out your GPU even if you use it occasionally for other tasks.
4. **Can I rent out multiple GPUs through FluxCore?** Yes, you can connect multiple machines with multiple GPUs to the FluxEdge Marketplace.
5. **Is there a limit to the number of hours I can rent out my GPU each month?** There are no specifications for maximum rental duration. The machine can be rented as long as the customer pays for it.
6. **What can I expect in terms of earnings stability as a FluxCore host?** Earnings stability can vary based on demand and the specifications of your machine. FluxCore aims to provide a consistent income stream for hosts.
