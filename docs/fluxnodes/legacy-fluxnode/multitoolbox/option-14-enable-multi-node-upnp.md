# Option 14 – Enable Multi‑Node (UPnP)

Configures UPnP port mappings to run up to 8 FluxNodes on a single public IP.

***

### Prerequisites

* Router with UPnP enabled
* Static local IP for each node

***

### How to Use

1. Select **14**.
2. Toggle UPnP **On**.
3. Enter desired **API port** (choose from 16127, 16137, …, 16197).
4. Verify or manually set your router’s gateway IP if prompted.
5. Repeat for each additional node instance.
