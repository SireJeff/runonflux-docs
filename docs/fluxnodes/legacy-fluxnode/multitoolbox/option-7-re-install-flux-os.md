# Option 7 – Re‑Install Flux OS

Performs a clean reinstall of FluxOS components without wiping your config file.

***

### When to Use

* You suspect corrupted FluxOS files
* An upgrade failed and left your OS in inconsistent state

***

### How to Use

1. Ensure you have your install config from **Option 6**.
2. Select **7**.
3. Confirm and let the script teardown and rebuild FluxOS.
