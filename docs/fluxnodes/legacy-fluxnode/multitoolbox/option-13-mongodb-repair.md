# Option 13 – MongoDB Repair

Attempts to fix or rebuild the MongoDB instance that FluxOS uses for app data.

***

### Modes

* **Soft Repair:** Quick rebuild of indexes
* **Hard Repair:** Full MongoDB reinstallation and document re‑indexing (may take up to 1 hour)

***

### How to Use

1. Select **13**.
2. Choose **Soft** first; if issues persist, repeat with **Hard**.
