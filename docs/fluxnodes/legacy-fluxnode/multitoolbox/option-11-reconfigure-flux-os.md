# Option 11 – Reconfigure Flux OS

Re‑runs key FluxOS setup routines—networking, SSH, notifications—without a full OS reinstall.

***

### What You Can Change

* ZelID
* KDA (Kadena) address
* Development vs. Mainnet channel toggle
* Thunder mode (enable storage benchmarking)

***

### How to Use

1. Select **11**.
2. Navigate through prompts to adjust settings as needed.
