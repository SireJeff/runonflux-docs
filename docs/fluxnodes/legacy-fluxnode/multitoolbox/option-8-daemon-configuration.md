# Option 8 – Daemon Configuration

Corrects or updates your node’s on‑chain credentials if you mistyped them initially.

***

### Use Cases

* Invalid Identity Key, TX ID, or Output Index errors in logs
* Migrated collateral to a new wallet

***

### How to Use

1. Select **8**.
2. Paste the correct **Identity Key**, **Collateral TX ID**, and **Output Index**.
3. The script updates your daemon config and restarts services.
