# Option 3 – Analyzer & Fixer

Runs a suite of diagnostics and attempts automated fixes for common node problems.

***

### Use Cases

* Node won’t sync or respond
* Unexpected daemon errors
* General health check before seeking support

***

### How to Use

1. From the menu, select **3**.
2. Review the output for errors (e.g., missing dependencies, daemon crashes).
3. Follow any “Help” tips shown, or share the full output with support/community.
