# Option 4 – Install Watchdog

Installs or reconfigures the **Flux Watchdog** service, which self‑heals, auto‑updates, and pushes alerts.

***

### Prerequisites

* Discord Webhook URL
* Discord User ID (for pings)

***

### How to Use

1. Select **4** to start the guided Watchdog setup.
2. Choose whether to enable auto‑updates for FluxOS, `fluxd`, and benchmarks.
3. Paste your Discord Webhook and User ID when prompted.
4. Confirm and finish—Watchdog will run under **pm2**.
