# 5. Configure Notifications

Stay informed about your FluxNode’s status by setting up optional alerts. You can configure notifications now or skip and return later.

***

### Supported Platforms

* **Discord**\
  Enter a Discord webhook URL to post alerts in your chosen channel.
* **Telegram**\
  Enter your Telegram bot token and chat ID to receive messages directly.

> ℹ️ You may skip this step and configure notifications at any time via the FluxNode web UI.
