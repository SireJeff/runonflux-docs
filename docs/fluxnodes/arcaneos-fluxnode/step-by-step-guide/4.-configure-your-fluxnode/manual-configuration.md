# Manual Configuration

To configure your FluxNode manually, you’ll first gather all necessary wallet credentials and then enter them into the Arcane OS interface. Follow these steps:

***

### Step1: Gather Wallet Credentials

1. Open **ZelCore Wallet** → **FluxNode** → **Select Your Node** → **Edit**
   * Copy your **Private Key**, **Collateral TX ID**, and **Output Index**.

<img src="/.gitbook/assets/1 (2).png" alt=""/>

2. Go to **Apps** → **FluxID**, and copy your **Flux ID**.

<img src="/.gitbook/assets/2 (1).png" alt=""/>

**You should now have:**

* ✅ Private Key
* ✅ Collateral TX ID
* ✅ Output Index
* ✅ Flux ID

***

### Step2: Enter Node Details

1. In the FluxNode web UI, fill in the four fields you gathered.
2. After pasting the Collateral TX ID, the **Output Index** will auto‑populate.
3. The TX ID is validated against the network to prevent duplicates.
   * ⚠️ If it’s already in use, you may still proceed—just be aware of potential conflicts.
4. Click **Next** to proceed to [**Configure Notifications**](../5.-configure-notifications.md) in the next chapter.
