# Configure via ZelCore Token

Using a ZelCore token allows you to import all node settings in one step. Generate the token in your wallet, then paste it into Arcane OS

***

### Step1: Generate Your Token

1. Open **ZelCore Wallet** → **FluxNode** → **Select Your Node** → **Add to Flux Storage**.
2. Copy the **two‑word token** (e.g., `fire-tiger`).

<img src="/.gitbook/assets/1.png" alt=""/>

<img src="/.gitbook/assets/2.png" alt=""/>

***

### Step2: Import into ArcaneOS UI

1. Paste your token into the **Token** field in the Arcane OS setup screen.
2. Click **Import**.
   * ✅ All node fields will auto‑populate.
   * ✅ Any previous notification settings will be restored.
3. Proceed to [**Configure Notifications**](../5.-configure-notifications.md) in the next chapter.
