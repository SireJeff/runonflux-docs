# Configure via SSP Token

Similar to ZelCore, you can import configuration via an SSP token. Generate it in the SSP Wallet, then import

***

### Step1: Generate Your Token

1. Open **SSP Wallet** → **Nodes** tab → **Add to Flux Storage**.
2. Confirm the prompt and copy the **storage phrase**.

<img src="/.gitbook/assets/1 (1).png" alt=""/>

<img src="/.gitbook/assets/3.png" alt=""/>

***

### Step2: Import into ArcaneOS UI

1. Paste the SSP phrase into the **Token** field in the Arcane OS setup screen.
2. Click **Import**.
   * ✅ All node fields and settings will auto‑fill, just like with ZelCore.
3. Proceed to [**Configure Notifications**](../5.-configure-notifications.md) in the next chapter.
