# Marketplace

FluxCloud features a **dedicated application marketplace**, offering plug-and-play ready templates to deploy applications in just a few clicks.

Whether you're looking to run a game server, launch a blockchain node, or deploy a full web app stack, the marketplace provides a seamless experience across a wide range of categories.

***

### Overview

The FluxCloud Marketplace includes a growing collection of prebuilt application templates across multiple categories:

* **Games** – Palworld, Minecraft, Enshrouded, and more
* **Blockchain** – Run masternodes, validators, and other web3 infrastructure
* **Productivity** – Self-hosted apps for development, note-taking, monitoring, and more
* **Front-end** – Static sites, UIs, dashboards
* **Masternodes** – Preconfigured node templates for popular crypto projects

### Browsing the Marketplace

You can find applications in the Marketplace using either of the following:

* **Category Filters** – Quickly narrow down apps by type (e.g., Blockchain, Games, Productivity)
* **Search Bar** – Instantly locate specific apps by name

From the [Marketplace Page](https://cloud.runonflux.io/marketplace), just scroll or use filters to explore what's available.

***

### One-Click Deployment

Each app listing includes:

* A description of what it does
* Input fields for any custom settings
* A single **Install** button to launch the app

Once you’ve filled out the details and chosen a plan, your app will be deployed across the FluxCloud and live within 10-30 min.
