# Rewards

The Rewards tab provides a breakdown of FLUX token earnings for each FluxNode tier across the network.

***

### What You'll See

This dashboard displays the following information:

#### FluxNode Rewards

* **Collateral**: The collateral required for each FluxNode tier:
  * Cumulus: 1,000 FLUX
  * Nimbus: 12,500 FLUX
  * Stratus: 40,000 FLUX
* **Rewards**: The estimated rewards per node tier, including:
  * **Per Day**: Native FLUX rewards + FLUX from Parallel Assets
  * **Per Week**: Native FLUX rewards + FLUX from Parallel Assets
  * **Per Month**: Native FLUX rewards + FLUX from Parallel Assets

#### Historical Price Chart

* A one-month time-series graph showing the historical FLUX token price in USD.
* Hover over the graph to view the exact price at a specific date and time.
