# List

The List tab offers a searchable and paginated view of all FluxNodes operating on the network.

***

### What You'll See

This dashboard displays the following data columns for each FluxNode:

* **IP Address**: The public IP of the FluxNode.
* **Address**: The wallet address associated with the FluxNodeâ€™s collateral.
* **Country**: The reported country location of the FluxNode.
* **Provider**: The hosting provider of the FluxNode.
* **Last Paid**: The timestamp of the last reward payment received by the FluxNode.
* **Tier**: Indicates the FluxNode tier: Cumulus, Nimbus, or Stratus.

Additional Features:

* **Pagination**: Easily navigate between pages to browse the full list of FluxNodes.
* **Search/Filter**: Use the search bar to filter results by any column value, such as IP address or wallet address.
