# Global Apps

The **Global Apps** section displays all applications currently running on the FluxCloud network.

***

### Overview of Tabs

There are two tabs:

* **Active Apps** – All applications actively running on FluxCloud
* **Marketplace Deployments** – Applications deployed from the marketplace.

<img src="/.gitbook/assets/Global Apps.png" alt=""/>

***

### What You'll See

For each application, the following details are shown:

* **App Name**
* **App Description**
* **Resources and Instances** – CPU, RAM, storage, and number of instances.
* **Subscription Period** – Number of days remaining on the current subscription.
* **Visit App** – A button to open the application's front-end (if available).

Additional features include:

* A **search bar** to find applications deployed on the FluxCloud network.
* A **selector** to set how many entries per page you want to see.
