# Register New App

FluxCloud makes it easy to deploy and manage applications across a global, distributed network. This guide walks you through launching a custom application on FluxCloud using the **Register New App** tab.

**How To Access**

> **Applications → Register New App**

Extend the **Applications** menu tab, then click **Register New App**.

***

### Overview of Sections

You'll configure all necessary settings across four main sections:

* **Details**
* **Component**
* **Validate & Pay**
